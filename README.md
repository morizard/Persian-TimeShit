# ⏰ ردیاب ساعات کاری پیشرفته

سیستم جامع مدیریت حضور و غیاب، ثبت شرح کار (Timesheet)، محاسبه حقوق و دستمزد، مدیریت اضافه‌کاری و گزارش‌گیری مالی.

> یک نرم‌افزار کاملاً فارسی، بدون نیاز به سرور یا پایگاه داده، که تنها با یک فایل HTML اجرا می‌شود.

---

## ✨ امکانات

### 📅 مدیریت حضور و غیاب

* ثبت ساعت ورود و خروج
* ثبت سریع با ساعت فعلی
* انتخاب تاریخ شمسی
* ویرایش اطلاعات روزهای گذشته
* ثبت وضعیت روز:

  * حضوری
  * دورکاری
  * مرخصی
  * تعطیل رسمی

---

### 📝 تایم‌شیت (شرح کار)

* ثبت نامحدود وظایف برای هر روز
* تعیین ساعت شروع و پایان هر فعالیت
* محاسبه زمان صرف‌شده برای هر وظیفه
* مشاهده تایم‌لاین کامل ماه
* کپی گزارش کامل تایم‌شیت

---

## 🚀 اجرای پروژه

هیچ وابستگی یا نصب خاصی نیاز نیست.

کافی است فایل زیر را اجرا کنید:

```text
index.html
```

یا پروژه را روی هر وب‌سرور ساده (Apache، Nginx، GitHub Pages و...) قرار دهید.

## 💾 ذخیره اطلاعات

تمام اطلاعات داخل **LocalStorage** مرورگر ذخیره می‌شوند.

در صورت پاک شدن داده‌های مرورگر، اطلاعات حذف خواهند شد؛ بنابراین پیشنهاد می‌شود به صورت دوره‌ای از بخش پشتیبان، فایل JSON تهیه کنید.
